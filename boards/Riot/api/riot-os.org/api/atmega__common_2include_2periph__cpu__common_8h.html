    )    ((s2x << 2) | (pr1 << 1) | pr0)

    SPI speed selection macro.

    We encode the speed in bits 2, 1, and 0, where bit0 and bit1 hold the SPCR prescaler bits, while bit2 holds the SPI2X bit.
