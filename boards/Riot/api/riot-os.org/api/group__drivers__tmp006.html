    Detailed Description

    Driver for the Texas Instruments TMP006 sensor.

    The TI TMP006 (Infrared Thermopile Contactless Temperature Sensor) measures the temperature of an object without need of direct contact with the object. After initialization the sensor can be set active for periodic measurements.
    The conversion duration depends on oversample ratio. The oversample ratio can be determined by sensor initialization. The target object temperature can be calculated as follows:

    ◆ TMP006_USE_LOW_POWER

    #define TMP006_USE_LOW_POWER   (0)

    Default low power mode.

    If set to 0, the device will be always-on If set to 1, the device will be put in low power mode between measurements. This adds a TMP006_CONVERSION_TIME us delay to each measurement call for bringing the device out of standby.

    ◆ TMP006_USE_RAW_VALUES

    #define TMP006_USE_RAW_VALUES   (0)

    Default raw value mode.

    If set to 0, measurements will be converted to Celsius. If set to 1, raw adc readings will be returned.
