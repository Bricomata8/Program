    Detailed Description

    Common interface to the software PRNG.

    Various implementations of a PRNG are available:

    • Tiny Mersenne Twister (default)
    • Mersenne Twister
    • Simple Park-Miller PRNG
    • Musl C PRNG
    • Fortuna (CS)PRNG
