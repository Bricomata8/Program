    Detailed Description

    TFTP support library.

    The TFTP module add's support for the TFTP protocol. It implements the following RFC's:

    • https://tools.ietf.org/html/rfc1350 (RFC1350 The TFTP Protocol (Revision 2)
    • https://tools.ietf.org/html/rfc2347 (RFC2347 TFTP Option Extension)
    • https://tools.ietf.org/html/rfc2348 (RFC2348 TFTP Blocksize Option)
    • https://tools.ietf.org/html/rfc2349 (RFC2349 TFTP Timeout Interval and Transfer Size Options)
