    ◆ GPIO_MODE

    #define GPIO_MODE (   mode,
      cnf,
      odr 
    )    (mode | (cnf << 2) | (odr << 4))

    Generate GPIO mode bitfields.

    We use 4 bit to determine the pin functions:

    • bit 4: ODR value
    • bit 2+3: in/out
    • bit 1: PU enable
    • bit 2: OD enable
